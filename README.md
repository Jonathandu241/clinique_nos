# Clinique NOS

Socle Next.js pour une application de gestion de rendez-vous clinique.

Nom technique du projet: `clinique-nos`

## Scripts

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run typecheck`
- `npm run format`

## Statut

Le projet contient pour l'instant le socle applicatif uniquement.
